# Build Outputs

This directory contains exported game builds. Build artifacts are generated by the CI/CD pipeline (`.github/workflows/upload-itch.yml`) and are git-ignored.

## Platforms

| Directory | Platform | Output |
|-----------|----------|--------|
| `windows/` | Windows Desktop | `WordCannon.exe` |
| `linux/` | Linux | `WordCannon.x86_64` |
| `macos/` | macOS | `WordCannon.zip` |
| `web/` | Web (HTML5) | `index.html` |
| `android/` | Android | `WordCannon.apk` |
| `ios/` | iOS | `WordCannon.ipa` |
